# **App Name**: CodeNote

## Core Features:

- Topic Creation: Allow users to create new topics or categories for organizing their code notes.
- Note Taking: Provide a rich text editor (like markdown editor) within each topic for writing code snippets and notes.
- Note Saving: Enable users to save their notes and code snippets under the corresponding topics, backed by a Firestore database.
- Organized Display: Display topics in a sidebar (left or right) for easy navigation. Each topic shows a list of note titles.
- Code Highlighting: Automatically detect and apply syntax highlighting for different coding languages within the note editor using generative AI tool.
- Full-Text Search: Implement a full-text search feature to find specific code snippets or notes across all topics.

## Style Guidelines:

- Primary color: Soft, desaturated blue (#94B49F) to promote calm focus.
- Background color: Light beige (#F5F5DC) for a neutral, distraction-free writing environment.
- Accent color: Warm orange (#E6A969) for interactive elements like buttons and links, for contrast.
- Headline font: 'Space Grotesk', a proportional sans-serif for headers.
- Body font: 'Inter', a grotesque-style sans-serif for comfortable reading of longer text.
- Code font: 'Source Code Pro' monospaced font for displaying code snippets.
- Simple, line-based icons for topic categories and common actions (save, edit, delete).
- Sidebar navigation on the left with collapsible topics for easy access to notes.
- Subtle transitions and animations when navigating between topics and notes to provide feedback and maintain user orientation.
